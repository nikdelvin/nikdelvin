### Hi there 👋

### This is my portfolio repository.

#### The current version is deployed at nikdelvin.com.

#### The repository consists of two projects:
```
1. Pocket Space - an attempt to understand the generation of shaders in JS and create a randomly generated small universe consisting of galaxies, black holes, stars and planets.
1.1. Every object in the universe is randomly generated from the surface to the color palette
1.2. The main project code is /src/pages/PocketSpace.jsx
1.3. You can study the current version at nikdelvin.com/poketspace
```
```
2. Market - a test task for creating a small demo Android application in conjunction with React and Node.js, which is a store with a catalog and a login form.
2.1. An admin panel has also been added to add new products to the catalog, delete and edit products. As well as managing user accounts.
(Login to the admin panel according to the data: login admin, password admin.)
2.2. Main project code - /src/pages/Market .jsx
2.3. You can study the current version at nikdelvin.com/market
2.4. Android application for the Market project - /src/android/nikdelvin, /src/android/app -debug.apk (built APK file)
```
